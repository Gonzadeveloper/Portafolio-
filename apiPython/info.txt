Mi nombre es Gonzalo Cayssials. Soy desarrollador web argentino con base en La Plata. Actualmente estudio Ingeniería en Sistemas de la Información en la UTN desde 2023. También estudié en el bootcamp Henry y me formé como tester QA en 2023.

He desarrollado múltiples proyectos web como desarrollador full stack. Me especializo en JavaScript, TypeScript, React, Node.js, PostgreSQL y MongoDB.

Uno de mis proyectos destacados es AppCar, una aplicación móvil desarrollada en React Native para combatir el fraude en la venta de autos usados. Utiliza OCR, web scraping con Python y verificación de kilometraje cruzando datos de fuentes públicas. Desarrollé este proyecto completamente por mi cuenta.

También desarrollé FluentWay, una landing page con sistema de login/registro y packs de cursos de inglés. Usé React para el frontend y Node.js con Express y PostgreSQL en el backend. Implementé autenticación con JWT.

Otro proyecto importante es Pedilo, una app de viajes al estilo Uber. Me encargué de toda la lógica de backend, incluyendo WebSockets para ubicación en tiempo real y un algoritmo para encontrar el conductor más cercano.

Desarrollé CashIn, una app de finanzas personales con IA. Me ocupé del backend completo, autenticación con OAuth y diseño de base de datos en PostgreSQL. También integré rutas y lógica de negocio con Node.js y Express.

También hice una app de IA local llamada Chat IA, que utiliza LLaMA 3 y tecnologías como WebGPU y Web Workers para correr modelos directamente en el navegador, sin backend.

Mi experiencia también incluye liderar el frontend del e-commerce Electro Emporium, usando React, Redux, TypeScript y Bootstrap. Me encargué de las rutas, filtros, carrito, y conexión con MercadoPago.

Mi primer proyecto académico fue PI Drivers, donde desarrollé todo el stack PERN para listar, filtrar y crear pilotos de Fórmula 1. Me ayudó a entender en profundidad las bases del desarrollo full stack.

Además, trabajé como backend developer en Lumia SRL en un sistema de autocoaching financiero con IA. Implementé una API robusta, autenticación y recomendaciones personalizadas.

Doy clases particulares sobre el stack PERN, y genero contenido en TikTok e Instagram sobre desarrollo web para ayudar a otros a aprender.

Mis habilidades incluyen comunicación, liderazgo, resolución de problemas y aprendizaje continuo. Me interesa trabajar en equipos que desafíen mi capacidad técnica y me permitan crecer como desarrollador.
